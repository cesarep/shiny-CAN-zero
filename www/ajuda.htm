  • Número e variável devem ter uma multiplicação. Ex.: 3x → 3*x;
  • Exponencial: Base e: exp(x), Base b: b^x;
  • Raiz quadrada: sqrt(x);
  • Logaritimo: Base e: log(x), Base 10: log10(x), Base b: log(x, b)
  • Trigonométricas: sin(x), cos(x), tan(x);
  • Arcos de trigonométricas: asin(x), acos(x), atan(x);
  • Hiperbólicas: sinh(x), cosh(x), tanh(x);
  • Hiperbólicas inversa: asinh(x), acosh(x), atanh(x);
  • Módulo: Mod(x);
